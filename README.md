# Atividades de Programação Web (Front-End) 🚀

Repositório destinado ao armazenamento dos materiais e atividades práticas desenvolvidas na disciplina de **Programação Web (Front-End)**, do curso de graduação em **Análise e Desenvolvimento de Sistemas** da **UniFAFIRE**.

## 📌 Sobre 
O objetivo deste repositório é documentar a evolução técnica nos estudos de interface, desde a estruturação semântica até a interatividade com JavaScript.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de páginas.
* **CSS3:** Estilização, Box Model e Layouts com Flexbox.
* **JavaScript (ES6+):** Manipulação do DOM e eventos de interatividade.
